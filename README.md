<p align="center">
  Repositório de estudos voltado aos primeiros passos no aprendizado de <strong>Git</strong> e <strong>versionamento de código</strong>.
</p>



## 📚 Sobre o repositório

Este repositório foi criado com o objetivo de registrar minha evolução nos estudos sobre **Git**, reunindo anotações, comandos, testes e exercícios práticos desenvolvidos durante o processo de aprendizagem.

A proposta é utilizar este espaço como apoio para fixar os principais conceitos de versionamento e também como material de consulta para revisões futuras.



## 🎯 Objetivo

O principal objetivo deste repositório é fortalecer a base no uso do **Git**, compreendendo na prática como funciona o controle de versões, a organização de alterações em arquivos e a utilização de recursos essenciais no dia a dia do desenvolvimento.

---

## 🛠️ Conteúdos abordados

Neste repositório, serão praticados temas como:

- Inicialização de repositórios
- Configuração do Git
- Adição e remoção de arquivos
- Commits
- Histórico de versões
- Branches
- Merge
- Resolução de conflitos
- Boas práticas com Git
